# Claude Cowork для PredictHub — Полный гайд по настройке и автоматизации

## ЧТО ТАКОЕ CLAUDE COWORK

Claude Cowork — это AI-агент встроенный в Claude Desktop. Он может:
- Управлять твоим компьютером (кликать, печатать, открывать приложения)
- Работать с файлами (читать, создавать, редактировать)
- Открывать браузер и делать действия на сайтах
- Выполнять задачи по расписанию (ежечасно, ежедневно, еженедельно)
- Принимать задачи с телефона (Dispatch) и делать на компе

---

## ЧТО НУЖНО ДЛЯ СТАРТА

### Железо и софт
| Что | Требование |
|-----|-----------|
| ОС | macOS (обязательно) или Windows x64 |
| Подписка | Claude Pro ($20/мес) или Max ($100/мес) |
| Приложения | Claude Desktop (последняя версия) |
| Для Dispatch | Claude mobile app (iOS/Android) |
| Интернет | Постоянное подключение |
| Комп | НЕ должен уходить в сон пока работает задача |

### Первоначальная настройка (30 минут)

**Шаг 1: Установить Claude Desktop**
- Скачать с claude.com
- Войти с аккаунтом Pro/Max

**Шаг 2: Создать рабочую папку проекта**
- Указать папку `C:/Projects/полимаркет/site/` как рабочую
- Claude получит доступ к чтению и записи файлов в этой папке

**Шаг 3: Создать контекстные файлы (КРИТИЧНО)**
Claude читает эти файлы автоматически при каждом запуске задачи:

Файл `about-me.md` в рабочей папке:
```markdown
# About Me
I'm Alex — prediction market trader since 2023, building PredictHub affiliate site.
My priorities:
1. Keep site content fresh and accurate (update weekly)
2. Grow organic traffic through SEO
3. Post daily on X/Twitter with market analysis
4. Monitor referral earnings on Polymarket and Kalshi
5. Track competitor positions in Google

# My Accounts
- Polymarket: [login info reference]
- Kalshi: [login info reference]
- Google Search Console: connected
- X/Twitter: @predicthub
- Telegram: t.me/predicthub

# Important Rules
- NEVER make up data. Always check real sources.
- Content must sound human, not AI-generated
- Update "Updated: March XX, 2026" dates when editing pages
- All affiliate links use rel="nofollow sponsored"
```

Файл `project-context.md`:
```markdown
# PredictHub Project
SEO affiliate site for prediction markets (Polymarket, Kalshi).
Revenue model: referral commissions (Polymarket 30% RevShare, Kalshi $25/ref).

# Site Structure
/index.html — Homepage (Best Prediction Markets 2026)
/polymarket/promo-code.html — MONEY PAGE #1
/kalshi/promo-code.html — MONEY PAGE #2
/compare/polymarket-vs-kalshi.html — Comparison
/countries/ — Country guides (India, Brazil, etc.)
/guides/ — Educational content

# Key Metrics to Track
- Google positions for: "polymarket promo code", "kalshi promo code", "polymarket vs kalshi"
- Weekly referral signups and earnings
- Site traffic from Google Search Console
- Competitor movements in SERP
```

**Шаг 4: Подключить коннекторы**
- Google (для Search Console, Gmail)
- Slack/Telegram (для уведомлений)
- Chrome extension (для браузерных задач)

### ⚠️ РАБОТАЕТ С ОГОВОРКАМИ (надёжность 50-70%)

| Задача | Проблемы | Решение |
|--------|----------|---------|
| **Публикация постов в X/Twitter** | Может ошибиться с UI, медленно | Подготовить текст → одобрить → Claude публикует |
| **Заполнение форм на чужих сайтах** | Капчи, нестандартные формы | Ручной контроль на сложных формах |
| **Аналитика в GA4/GSC** | Сложные дашборды, много кликов | Лучше через MCP-сервер (API), а не через UI |
| **Создание изображений для соцсетей** | Ограниченные возможности | Подготовить шаблоны в Canva, Claude заполняет |

### ❌ НЕ МОЖЕТ / НЕ СТОИТ ДЕЛЕГИРОВАТЬ

| Задача | Почему |
|--------|--------|
| **Вести живые стримы** | Claude не может говорить голосом в реальном времени, не может быть "ведущим" стрима |
| **Отвечать в чате стрима** | Нет интеграции с YouTube/Twitch чатом в реальном времени |
| **Торговать на Polymarket автоматически** | Computer Use слишком медленный для трейдинга (каждое действие = скриншот + решение). Для трейдинга используй Polymarket API или бота типа Polystrat |
| **Работать 24/7 без перерыва** | Комп должен быть включён, задачи не запускаются если комп спит |
| **Параллельные задачи** | Dispatch обрабатывает только 1 задачу за раз |
| **Работать на закрытых сайтах с 2FA** | Не может пройти двухфакторную аутентификацию |

### Расписание задач (Scheduled Tasks)

#### ЕЖЕДНЕВНО (Daily)

**Задача 1: Утренний брифинг по рынкам**
```
Промпт: "Зайди на polymarket.com и kalshi.com. Найди 3 самых
горячих рынка на каждой платформе (по объёму торгов за 24ч).
Сохрани список в /reports/daily-markets.md с датой.
Подготовь короткий текст для поста в X/Twitter (макс 280 символов)
про самый интересный рынок."

Частота: Daily, 9:00
```

**Задача 2: Проверка работоспособности сайта**
```
Промпт: "Открой каждую страницу нашего сайта из sitemap.xml.
Убедись что все загружаются без ошибок. Если что-то сломано —
запиши в /reports/site-health.md."

Частота: Daily, 10:00
```

#### ЕЖЕНЕДЕЛЬНО (Weekly)

**Задача 3: SEO-мониторинг (Понедельник)**
```
Промпт: "1. Открой Google Search Console для нашего сайта.
2. Выгрузи данные за последнюю неделю: клики, показы, средняя позиция.
3. Сравни с прошлой неделей.
4. Для запросов 'polymarket promo code', 'kalshi promo code',
   'polymarket vs kalshi' — запиши текущие позиции.
5. Сохрани отчёт в /reports/seo-weekly-[дата].md
6. Если какая-то страница упала больше чем на 3 позиции —
   отметь как АЛЕРТ."

Частота: Weekly (Monday), 10:00
```

**Задача 4: Обновление контента (Среда)**
```
Промпт: "1. Зайди на polymarket.com — проверь текущий бонус,
   объём торгов, количество стран.
2. Зайди на kalshi.com — проверь текущий бонус, промокоды.
3. Обнови цифры на страницах:
   - /polymarket/promo-code.html
   - /kalshi/promo-code.html
   - /compare/polymarket-vs-kalshi.html
4. Поменяй 'Updated: March XX, 2026' на текущую дату.
5. Обнови sitemap.xml с новыми датами lastmod."

Частота: Weekly (Wednesday), 11:00
```

**Задача 5: Мониторинг конкурентов (Пятница)**

**Задача 6: Подготовка контента для X/Twitter (Воскресенье)**
```
Промпт: "На основе /reports/daily-markets.md за эту неделю,
```

## СТРИМЫ И КОНТЕНТ — РЕАЛИСТИЧНЫЙ ПОДХОД

### Что Claude НЕ может делать со стримами:
- ❌ Быть ведущим стрима (нет голоса)
- ❌ Отвечать в чате в реальном времени
- ❌ Торговать в прямом эфире
- ❌ Управлять OBS/StreamYard

### Что Claude МОЖЕТ делать ДЛЯ стримов:

**ДО стрима:**
- Исследовать горячие рынки и подготовить список тем для обсуждения
- Написать скрипт/план стрима
- Создать overlay графику с данными рынков
- Подготовить слайды с аналитикой

**ПОСЛЕ стрима:**
- Взять запись стрима → сгенерировать текстовую расшифровку
- Нарезать ключевые моменты для YouTube Shorts / TikTok
- Написать пост-описание для YouTube
- Создать статью на сайт по теме стрима (SEO + контент)

### Альтернатива стримам с AI:

**Вариант 1: Записанные видео с AI-голосом**
- Claude пишет скрипт → ElevenLabs/LMNT озвучивает → ты монтируешь
- Дешевле и быстрее стримов, можно делать 3-5 видео в неделю

**Вариант 2: Screen-recording гайды**
- Claude готовит сценарий → ты записываешь экран с озвучкой
- "Как зарегистрироваться на Polymarket за 5 минут" — просто и эффективно

**Вариант 3: AI-торговля как контент**
- Запустить Polystrat (AI-бот для Polymarket)
- 14 из 20 топ-кошельков на Polymarket — уже боты
- Документировать результаты → контент для YouTube + блога

## ТРЕЙДИНГ — ОТДЕЛЬНАЯ ТЕМА

Claude Cowork НЕ подходит для трейдинга (медленный, 1 действие за раз).
Но есть специализированные инструменты:

| Инструмент | Что делает | Стоимость |
|-----------|-----------|-----------|
| **Polystrat** | AI-агент для торговли на Polymarket, 4200+ сделок/мес | Бесплатный (open) |
| **Polymarket API** | Программный доступ к торговле | Бесплатный |
| **Собственный бот** | Python + Polymarket API, Claude Code может написать | Время на разработку |

Один кошелёк сделал $300 → $400,000 за месяц через арбитраж
на коротких крипто-контрактах. Это реальные деньги.
